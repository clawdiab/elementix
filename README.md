# Elementix

A Web Components design system built with TypeScript.

## Components

- `<elx-button>` - Button component
- `<elx-input>` - Input component
- `<elx-card>` - Card component
- `<elx-badge>` - Badge component
- `<elx-alert>` - Alert component
- `<elx-avatar>` - Avatar component

## Usage

```html
<script type="module" src="https://unpkg.com/elementix/dist/index.js"></script>

<elx-button variant="primary">Click me</elx-button>
```

## Development

```bash
npm install
npm run dev
npm test
npm run storybook
```
